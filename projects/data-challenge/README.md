# Data Challenge - Dataset Titanic

Notebook complet montrant les étapes clés d'un workflow de Data Science :
- Exploration et visualisation des données
- Nettoyage et gestion des valeurs manquantes
- Feature engineering
- Entraînement et évaluation de modèles ML
- Comparaison des performances

Modèle final : Random Forest / Gradient Boosting  
Objectif : prédire la survie des passagers du Titanic.
