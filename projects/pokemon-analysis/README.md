# Pokémon Marketing Analysis

Analyse du dataset Pokémon pour identifier des tendances et clusters marketing.

## Objectifs
- Comprendre les caractéristiques des Pokémon
- Identifier des groupes de Pokémon avec des traits similaires
- Générer des insights visuels et analytiques pour des applications marketing

## Compétences démontrées
- Exploration et nettoyage de données
- Visualisation avancée
- Feature engineering
- Segmentation / clustering
